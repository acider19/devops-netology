Для корневого каталога gitignore пока пустой, значит в коммит будет попадать все.
Для файлов в каталоге terraform не попадут в коммит:
- содержимое скрытой папки .terraform
- файлы с расширением .tfstate
- файлы, содержащие в какой-то из частей имени расширение .tfstate (например *.tfstate.backup)
- логи с информацией об аварийных завершениях (crash) процессов и вариации этих логов (допустим с указанием даты в имени файла)
- файлы с чувствительными переменными terraform с расширением .tfvars и вариант в формате json
- файлы с локальной перезаписью ресурсов override.tf, также вариант в формате json и любые файлы содержащие в названии _override и с расширениями .tf и .tf.json
- файл переходного состояния .terraform.tfstate.lock.info
- файлы настроек командной строки .terraformrc и terraform.rc
